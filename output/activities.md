### Matemática

#### Programação Linear: Modelagem, solução gráfica e introdução ao método simplex.
**Professor:** Maria Cristina Nogueira Gramani

Este encontro aborda a modelagem e decisão de como melhor usar os recursos limitados disponíveis em problemas de otimização linear (ou seja, nos casos quando a função objetivo e as restrições são de natureza linear). Encontraremos a solução de um PL usando a resolução gráfica e introduziremos o método simplex, um procedimento para solucionar problemas maiores de programação linear; bem como o uso do Solver do Excel e/ou Python.

---

#### Teoria: Otimização Linear
**Professor:** Maria Cristina Nogueira Gramani
**URL:** https://integrada.minhabiblioteca.com.br/reader/books/9788580551198/pageid/104

Faça a leitura do capítulo 4 (seções 4.1 e 4.2, páginas 105 a 113). Este autoestudo especifica o objetivo como uma função linear de certas variáveis e especificar as restrições aos recursos como igualdades ou desigualdades lineares nessas variáveis, tem-se um problema de programação linear. Programas lineares surgem em uma variedade de aplicações práticas e serão vistos que estes problemas podem ser solucionados usando a forma gráfica (em problemas pequenos) e o Método Simplex (para problemas reais).

#### Programação Linear: Método Simplex
**Professor:** Maria Cristina Nogueira Gramani

Este encontro aborda o método simplex, um procedimento para solucionar problemas de programação linear. Desenvolvido por George Dantzig em 1947, provou ser um método extremamente eficiente que é usado, com frequência, para solucionar problemas atuais de grande porte. Descreveremos as principais características do método simplex e executaremos o procedimento para solucionar qualquer modelo de programação linear que se encontra em uma forma-padrão.
